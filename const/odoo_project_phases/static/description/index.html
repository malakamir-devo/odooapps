Project Phases and Project Task Phases - Odoo App

Manage Project using Project Phases

This app allows your project team / users to manage your project and task by phases.


Main Features:
  • Allow your project team to set up phases of your project.
  • Allow your user to select the project phase on the project task form.
  • Project Phases (Phases) Smart Button has been added on project form to quick view.
  • The Project Phase field has been added to the project task form.
  • Allow you to select and group by project phase in project task report.
  • For more details please see the video and check below screenshots.
