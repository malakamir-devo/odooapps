Change Order Management for Job Contracting Projects

This app allows your project team to create and manage change orders in job costing, job contracting, and construction projects.


Main Features:

  • Allow Project / Site / Sales Users to create and manage change order requests for your projects.
  • Workflow : Draft -> Confirm -> Approve -> Customer Approve -> Done -> Create Quotation.
  • Change orders can be reviewed and approved by project managers.
  • Allow you to print change orders in PDF and also send them to your customer and the system stored it on open chatter on the request form.
  • Allow you to create sales quotations directly from the change order form.
  • Integrated with the Sales App as we are allowed to create sales quotes created from change orders.
  • Change order link smart button on the project, job order/work order, and customer form as shown.
  • For more details about job costing you can see the dependent app.https://apps.odoo.com/apps/modules/16.0/odoo_job_costing_management/
