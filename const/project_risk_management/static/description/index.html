Risk Management for Project and Project Tasks Odoo

Project Risk Management in Odoo

Project Task Risk Management in Odoo

This app allows your project team users and project managers to have features to manage risk on projects and project tasks including creation of risk Incidents from project and tasks.


Main Features:

  • Allow your project users to create Risks from the Risk menu under configuration.
  • Allow your managers to view Risk analysis of projects and tasks for reporting purposes.
  • Allow you to create and manage to set Risk type, risk category and risk response setup under configuration.
  • Risk incidents creation and management during project and task execution.
