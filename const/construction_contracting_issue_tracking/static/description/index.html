Construction and Contracting Issue Tracking Management System

Construction and Job Contracting Issue Tracking Management System in Odoo

This module allows you to manage construction and job contracting related issues. This app is an issue tracking system for construction and job contracting business.
This issue tracking system is for workers who work on site at construction and job contracting and allow them to raise any kind of issues related to jobs they are working on. Your customer can also use this system to raise support issues on construction and job contracting sites.

Main Features:

  • Your Customer / Construction Team can send issues requests from your website to my account / portal page and also attach documents while creating issues from the website.
  • Generation of unique issue numbers on submission and record it as an issue in the backend side.
  • The Customer / Construction Team can check the status of all issues submitted by him/her on my account portal page.
  • Allow your backend team to print a PDF report of issue.
  • Project / Issue: User / Technician can communicate using chatter and fill timesheets.
  • Project / Issue managers can close tickets and send bills to customers if paid support.
  • Customer/Team can give feedback and rating of the ticket.
  • Manage your issued tickets using assignments to an issue team.
  • Prepaid hours of customers for issues for that analytic account / project.
  • Allow you to create customer invoices from timesheets.
  • Allow you to create tasks on issues and fill timesheets from issue form.
  • Allow you to add invoice lines on issue form and create invoice from issue form.
  • Select timesheets for which you want to create customer invoices and click on the Make Invoice wizard action to create a customer invoice under the invoicing Menu.
  • Prepaid hours configuration on customer analytic account and show status of prepaid hours on issue ticket form.
  • Allow your user to add Time In and Time Out on timesheets.
  • Stages for issue management systems related to your construction and contracting business and workflow.
  • Issue Manager can set up types of issue subject under the configuration menu of issues.
  • Issue Manager can set up issue types under the configuration menu of issues.
  • Issue Manager can set up types of subjects under the configuration menu of issues.
