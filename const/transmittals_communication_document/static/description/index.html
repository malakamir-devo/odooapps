Transmittals Documents and Submittals Document for Construction and Job Contracting

For Construction and Job Contracting Project and Job Order / Tasks

Transmittals Documents and Submittals Document Management Odoo App

This app allows you to have submittals documents and transmittals documents feature for construction projects and job contracting projects.


Main Features:

  • Allow you to configure and manage Transmittal / Submittal Types.
  • Allow you to configure and manage Transmittal / Submittal Mediums.
  • Send by email button to send transmittal documents to your receiver(s).
  • Send by email button to send submittal documents to your receiver(s).
  • Allow your project team to print PDF Report for Transmittal documents and Submittal documents as shown.
  • Smart Button to quick view of Submittal / Transmittal Documents on Project Form.
  • Smart Button to quick view of Submittal / Transmittal Documents on Job Order / Work Order Form.
  • Definition:
    • Submittal Documents refers to documents, samples, files and any other information that should be delivered to site engineers, architects, project consultants, and others involved in a construction project, even before starting the project.
    • A transmittal document is a document that is sent along with other documents and files to the receiver. It allows every person to keep track of all communications that occur between everyone in the construction project. It provides proof that the documents, samples, or files have been sent, and records when they were sent and who they were sent to. If a question arrives to the project team, they can refer to the transmittal later on.
