Import Job Cost Sheet from Excel

Import Job Cost Sheet from Excel Odoo App

This app will allow you to import a Job Cost Sheet from excel file as shown in below screenshots and excel sample.


Main Features:

  • This app will allow you to import a Job Cost Sheet from excel.
  • Make sure you have xlrd Python libarary installed.
  • Sample Excel is available inside the sample folder in the module.
  • For more details please see below screenshots and watch the video.

Following fields can be imported from Excel Columns.

  • NAME : Name
  • PROJECT : Project for Job Cost Sheet
  • ANALYTIC ACCOUNT : Analytic Account
  • JOB ORDER : Job Order
  • CUSTOMER : Partner/Customer
  • DESCRIPTION : Description for Job Cost Sheet
  • SALE REFERENCE : Sale Reference for Job Cost Sheet
  • DATE : Date for Material Line Purchase
  • JOB TYPE : Job Type for Material/Labour/Overhead
  • PRODUCT : Product
  • DESCRIPTION : Description of Product
  • REFERENCE : Reference of Product
  • PLANNED QTY : Planned Qty
