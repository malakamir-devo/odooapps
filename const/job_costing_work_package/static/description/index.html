General Features:

Our module provides features of a work package where you can list out all your tasks or job orders into a single work package and send it by email.

A construction work package defines a logical and manageable division of work orders/job orders within the construction project.

So, A work package is a group of related work orders/job orders/tasks within a project. Because they look like projects themselves, they are often thought of as sub-projects within a larger project. Tasks are typically grouped into work packages based on geographical area, engineering discipline, technology, or the time needed to accomplish them.
