Scan and Upload Images Framework Module for Portal

Base Framework Module for scan and upload images on my account portal. This is only a framework module so if you want to add the feature of uploading photos/images on my account menus then you will have to create a new module by using this module framework. For more details please contact us before buying.

Main Features:

  • Framework for adding the feature of Scan and Upload Images/Photos on the portal.
  • The framework for to have the feature of uploaded images is stored as attachments and chatter.
  • Framework for preview of attachment in the portal.
