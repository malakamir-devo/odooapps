Waste Management for Construction and Job Contracting

Construction Waste Manage using Scraps and Reuse Pickings Creation

This module allows you to manage construction waste materials using scrap and reuse methods for your construction and job contracting business.


Main Features:

  • Allow job order users to add Waste Materials on Job Order form and based on that Create Waste Picking or Stock Scrap based on waste method selected.
  • Users can decide whether waste will be reused or scrap.
  • Using the create button on the Waste material tab system will automatically create scrap record (Odoo standard scrap order) or internal picking (Odoo standard) record based on method selected.
  • Basically allow you to manage your waste during job orders / work orders process during construction projects execution.
